## 제어문

### 반복문
```
프로그램 내에서 똑같은 명령을 일정 횟수만큼 반복하여 수행하도록 제어하는 명령문
프로그램이 처리하는 대부분의 코드는 반복적인 형태가 많으므로, 가장 많이 사용되는 제어문 중 하나

자바에서 사용되는 대표적인 반복문의 형태
1. while 문
2. do / while 문
3. for 문
4. Enhanced for 문
```
1. while 문
``` java
특정 조건을 만족할 때까지 계속해서 주어진 명령문을 반복 실행한다

while (조건식) {
    조건식의 결과가 참인 동안 반복적으로 실행하고자 하는 명령문;
}
```
