### 늦은 초기화
```
객체 초기화를 늦게 하는 것
```
늦은 초기화 방법
|lateinit|by lazy|
|:----:|:-----:|
|`var` 프로퍼티에서 사용 가능|`val` 프로퍼티에서 사용 가능|
|get(), set() `불가능`|get(), set() `가능`|
|null 허용안함|nullable|
|생성자에서 사용 불가능|생성자에서 사용 불가능|
